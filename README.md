## 📝 详细流程

### 第一步：环境准备

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1.1 | 安装 gh (GitHub CLI) | `brew install gh` |
| 1.2 | GitHub 授权 | `gh auth login` |
| 1.3 | 配置 Git 用户 | `git config --global user.name` / `user.email` |

### 第二步：代码开发

```bash
# 创建工作目录
mkdir myopencode
cd myopencode

# OpenCode 启动（可选，或直接由 AI 助手生成代码）
opencode
```

**AI 生成的贪吃蛇核心代码逻辑：**

```javascript
// 蛇的移动
const head = { x: snake[0].x + dx, y: snake[0].y + dy };
snake.unshift(head);

// 吃到食物
if (head.x === food.x && head.y === food.y) {
  score += 10;
  placeFood();
} else {
  snake.pop(); // 移动时移除尾部
}
```

### 第三步：GitHub 推送

```bash
# 初始化 Git
git init
git add .
git commit -m "feat: 添加贪吃蛇游戏"

# 推送到 GitHub（创建新仓库）
gh repo create snake-game --public --source=. --push

# 或推送到已有仓库
git remote add origin https://github.com/username/repo.git
git push -u origin main
```

### 第四步：Vercel 部署

```bash
# 安装 Vercel CLI
npm install -g vercel

# Vercel 登录
vercel login

# 部署（确保入口文件为 index.html）
mv snake.html index.html
vercel deploy --prod
```
